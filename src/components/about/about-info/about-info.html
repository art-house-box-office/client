About the project

This application is designed for independent cinema owners to be able to track thier own movie sales, and see trends develop as other cinema owners track their own data.

This application was created using Angular, hosted on Firebase, with the API running on Node.js hosted by Heroku backed by a MongoDB database provided by mLab. Thanks to all the open-source developers that have put in all the effort to make these projects possible so that we can make awesome projects!
